## 🌟 项目缘起

迎来了新的一年，我想记录打卡。

1.  **拒绝付费：** 去网上找 Excel 模板，发现稍微好看点的都要钱、要会员，WPS 甚至还需要“超级会员”。
2.  **拒绝妥协：** 语雀自带的在线 Excel 模板又太简单，无法满足我“既要美观又要自动化”的需求。
3.  **AI 协作：** 虽然我的 Excel 能力有限，但我有 AI！通过不断将我的想法描述给 AI，我们一起一点点磨出了这份能自动生成“联动看板”的脚本。

> **意外惊喜：** 在编写过程中发现，Python 的 `openpyxl` 库居然比我以前用过的某些 Java Excel 处理包（如 POI）还要强大且易用。

---

## 📅 思路演变与迭代历程

这个脚本的成型，是我不断给 AI “提需求、修 Bug”的进化史：

### **1. 解决体力活 (V1 - V2)**
*   **痛点：** 手动创建 12 个月、填日期、对齐排版简直是折磨。
*   **AI 赋能：** 代码一键生成 12 个月的空表格，基础日期对齐、周末自动填色。

### **2. 解决“重复录入” (V3)**
*   **痛点：** 每月都要手动复制一遍打卡事项，极其反人类。
*   **AI 赋能：** 引入“事项配置页”。只需在首页填一次，12 个月通过公式自动同步。

### **3. 数据可视化 (V4)**
*   **痛点：** 表格太长，必须翻到底部才能看到统计结果。
*   **AI 赋能：** 
    *   在顶部增加“仪表盘”区域，实时展示月度达成率、连续天数。
    *   引入“条件格式”：填入即变绿，空行自动隐藏，页面瞬间清爽。

### **4. 年度全局视角 (V5)**
*   **痛点：** 月份间跳来跳去，看不出全年的坚持轨迹。
*   **AI 赋能：** 
    *   新增“年度汇总看板”，打造 365 天数据热力图方块墙。
    *   自创“百分制评分逻辑”（达成率 70% + 活跃度 30%），给坚持一个量化反馈。

---

## ✨ 核心亮点

*   **🎛️ 一处修改，全局联动：** “事项配置页”一改，全年 12 个月自动同步。
*   **📈 极简看板：**
    *   **月度：** 自动计算进度，带对比分析。
    *   **年度：** 365 天足迹方热力墙，成就感爆棚。
*   **🎨 智能交互：**
    *   **自动填色：** 输入内容格子即变绿，正向反馈。
    *   **动态渲染：** 仅显示有事项的行，拒绝杂乱。
    *   **防误删：** 关键公式单元格自动锁定保护。

## 🛠️ 如何使用

1.  **环境准备：**
    ```bash
    pip install openpyxl
    ```
2.  **生成模板：**
    ```bash
    python generate_excel_v5.py
    ```
3.  **使用建议：** 上传至语雀、飞书文档或本地 Excel 使用，体验最佳。
