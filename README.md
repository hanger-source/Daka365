# 📊 Daka365: 365天打卡 Excel 模板生成脚本

![Excel](https://img.shields.io/badge/Tools-Excel%20%7C%20Python%20%7C%20AI-green)
![Year](https://img.shields.io/badge/Year-2026-blue)

> **背景：** 
> 迎来了新的一年，我想记录打卡。我不太想用专门的软件，想直接记在语雀的在线 Excel 上，但发现自带模板太简单。由于我的 Excel 能力有限，所以我尝试让 AI 帮我写代码，通过不断地把我的想法描述给它，一点点磨出了这份能自动生成“联动看板”的脚本。

---

## 📅 思路演变与迭代历程

这个脚本的成型过程，其实是我不断给 AI “提需求、修 Bug”的过程：

### **1. 解决体力活 (V1 - V2)**
*   **最初的想法：** 手动创建 12 个月、填日期、排版太累了。
*   **AI 做的：** 用代码一键生成 12 个月的空表格，把基础的日期对齐、周末填色搞定。

### **2. 解决“重复录入” (V3)**
*   **遇到的问题：** 我发现每换一个月，都要把几十个打卡事项重新复制一遍，太反人类。
*   **改进思路：** 弄一个“总控开关”。
*   **AI 做的：** 增加了“配置页”。只需在首页填一次事项，后面 12 个月通过公式自动同步。

### **3. 让数据“看得见” (V4)**
*   **遇到的问题：** 表格太长了，往下翻才能看到总数，不够直观。
*   **改进思路：** 把关键统计数据挪到顶上，像个简单的“仪表盘”。
*   **AI 做的：** 
    *   在每个月顶部加了固定区域，展示达成率、连续天数等指标。
    *   加入了“条件格式”，填了内容格子就变绿，没填事项的行自动隐藏，让页面清爽一点。

### **4. 想要“全局视角” (V5)**
*   **遇到的问题：** 想看看一整年的坚持情况，月份页之间跳来跳去很麻烦。
*   **改进思路：** 增加一个全年度的汇总视图。
*   **AI 做的：** 
    *   增加“年度汇总看板”，把全年的打卡数据聚合成一个热力图方块墙。
    *   加了一个简单的 100 分制逻辑（达成率占 70 分，活跃天数占 30 分），给自己的坚持打个分。

---

## ✨ 核心功能

*   **Sheet页联动：** 只需在“事项配置页”修改，全年 12 个月的表格自动更新。
*   **简单看板：** 
    *   **月度：** 自动计算达成率，带简单的进度对比和坚持天数统计。
    *   **年度：** 一个简单的 365 天热力方块墙，展示一年的足迹。
*   **基础交互：**
    *   **填入变色：** 格子输入内容后背景自动变绿，成就感正反馈。
    *   **动态显示：** 只有填了事项的行才有边框和背景，空行保持纯白。
    *   **填写保护：** 锁定了带公式的单元格，防止误删。

---

## 🛠️ 如何使用

1.  安装 Python 和 `openpyxl` 库：
    ```bash
    pip install openpyxl
    ```
2.  运行脚本：
    ```bash
    python generate_excel_v5.py
    ```
3.  把生成的 `.xlsx` 文件上传到语雀或在本地打开即可。
